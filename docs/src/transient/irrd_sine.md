# IRRD Sine — 正弦源 + 分压电阻 + 二极管

**测试文件**: `tests/test_trans_irrd_sine.rs`
**测试函数**: `test_trans_irrd_sine_1kHz_10us`, `test_trans_irrd_sine_1kHz_1us`

## 电路

```
    Isin (正弦)
    ┌──→──┐
    │     │
    0     1
    │     ├── R1(10Ω) ── 0
    │     │
    │     └── D1(→) ── 2 ── R2(1kΩ) ── 0
    │
    └────────────────────────────────┘
```

**程序化网表**:
```spice
Isin 0 1 SIN(0, 0.3, f)
R1 1 0 10
D1 1 2 1e-9 27
R2 2 0 1000
```

## 解析

在 IRD 基础上增加了输出端的分压电阻 R2 (1kΩ)。

- R2 与二极管串联，限制二极管电流
- 节点 2 的电压是二极管导通电压（约 0.7V）减去 R2 上的压降
- 测试多节点瞬态波形的正确性
