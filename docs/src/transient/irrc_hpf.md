# IRRC HPF — RC 高通滤波器频响

**测试文件**: `tests/test_trans_irrc_hpf.rs`
**测试函数**: `test_irrc_trans_hpf_1kHz` ~ `10kHz`

## 电路

```
    Isin (正弦)
    ┌──→──┐
    │     │
    0     1
    │     ├── R_shunt(1Ω) ── 0
    │     │
    │     └── C_series(0.032µF) ── 2 ── R_shunt2(1kΩ) ── 0
    │
    └──────────────────────────────────────────────────┘
```

**程序化网表**:
```spice
Isin 0 1 SIN(0, 10, f)
R1 1 0 1
C1 1 2 0.032e-6
R2 2 0 1000
```

## 解析

一阶 RC 高通滤波器，与 LPF 对称。截止频率相同 (~5kHz)。

| 测试 | 频率 | 预期 |
|------|------|------|
| 1 kHz | 远低于 \\(f_c\\) | 大幅衰减 |
| 2 kHz | 低于 \\(f_c\\) | 明显衰减 |
| 5 kHz | 等于 \\(f_c\\) | -3dB 衰减 |
| 10 kHz | 高于 \\(f_c\\) | 几乎无衰减 |
