# IRRC — RC 低通滤波器（DC 视角）

**测试文件**: `tests/test_irrc.rs`
**测试函数**: `test_irrc_dc`

## 电路

```
    I1 (1A)
    ┌──→──┐
    │     │
    0     1
    │     ├── R1(10Ω) ── 0
    │     │
    │     ├── R2(1mΩ) ── 2
    │                    │
    │                   C1(0.01µF)
    │                    │
    └──────────────────── 0
```

**网表**:
```spice
I1 0 1 1.0
r1 1 0 10
r2 1 2 1e-3
c1 2 0 0.01e-6
```

## 解析

DC 工作点分析中，电容相当于**开路**（后向欧拉离散化中 \\(G_{eq} = C/\\Delta t\\)，但 DC 求解没有时间步）。

- 电容不导通，R2-C1 支路无电流
- 全部 1A 流经 R1
- \\(V_1 = 1A \\times 10\\Omega = 10V\\)
- \\(V_2 = V_1\\)（因为支路无电流，R2 上无压降）
