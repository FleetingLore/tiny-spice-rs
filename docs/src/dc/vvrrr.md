# VRRR — 双电压源 + 电阻网络

**测试文件**: `tests/test_vvrrr.rs`
**测试函数**: `test_vvrrr`

## 电路

参考 Swarthmore 大学 MNA 示例 2。

```
         R1 (2Ω)
    0 ───┤R├─── 1
               │
    V1 (32V)   │
    2 ───┤V├───┘
    │          │
    ├──┤R├─────┤
    │  R3(8Ω)  │
    │          │
    ├──┤R├── 3 │
    │  R2(4Ω)  │
    │          │
    └──┤V├─────┘
       V2 (20V)
```

**网表**:
```spice
r1 0 1 2
V1 2 1 32
r2 2 3 4
r3 2 0 8
V2 3 0 20
```

## 解析

- 节点数：4（含地）
- 电压源数：2
- MNA 矩阵维度：6 × 6

这是 MNA 的标准示例——含有多个电压源，需要 B/C 块来处理电压源支路电流。

## 验证

```rust
assert_nearly(v[1], 8.0);   // 节点 1
assert_nearly(v[2], 24.0);  // 节点 2
assert_nearly(v[3], 20.0);  // 节点 3
```
