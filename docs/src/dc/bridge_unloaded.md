# Diode Bridge (Unloaded) — 二极管桥式整流（空载）

**测试文件**: `tests/test_dc_bridge_p_unloaded.rs`
**测试函数**: `test_diode_bridge_unloaded_10V`

## 电路

与 Loaded 版相同的桥式拓扑，但输出端**没有负载电阻**（开路）。

**网表**:
```spice
V1 1 2 10
V2 2 0 0
D1 1 3 1e-9 27
D2 4 1 1e-9 27
D3 2 3 1e-9 27
D4 4 2 1e-9 27
; 无负载电阻！
```

## 解析

空载桥式整流的关键特性：
- 输出端开路，没有电流流经负载
- 二极管仍然会形成导通路径
- 输出电压接近输入电压（仅减去二极管导通压降）

与带载版的区别：负载电阻为 1kΩ 时会分流，影响二极管的工作点。
