# V_D_VS_D — 电压源 + 正反二极管对

**测试文件**: `tests/test_v_d_vs_d.rs`
**测试函数**: `test_v_d_vs_d`

## 电路

```
    V1 (5V)
    ┌──┤V├──┐
    │       │
    0       1
    │       ├──┤D├──┐  D1: 正向
    │       │        │
    │       │        2
    │       │        │
    │       └──┤D├───┘  D2: 反向 (阳极=0)
    │
    └───────────────┘
            0
```

**网表**:
```spice
V1 1 0 5
D1 1 2 1e-9 27
D2 0 2 1e-9 27
```

## 解析

- 节点数：3
- 电压源数：1
- MNA 矩阵维度：4 × 4

V1 输出 5V，D1 正向导通（约 0.7V 压降），D2 反向偏置（开路）。

$$V_2 = V_1 - V_{D1} \\approx 5 - 0.73 = 4.27\\text{V}$$

```rust
assert_nearly(v[1], 5.0);
assert_nearly(v[2], 4.2689);
```
