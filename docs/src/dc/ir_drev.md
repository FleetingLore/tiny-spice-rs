# IR_DREV — 反向偏置二极管

**测试文件**: `tests/test_ir_drev.rs`
**测试函数**: `test_ird_rev`

## 电路

```
    I1 (3A)
    ┌──→──┐
    │     │
    0     1
    │     ├──┤R├──┐
    │     │ 10Ω  │
    │     │      │
    │     ├──┤D├──┤
    │     │  反向  │
    └─────┘      │
          0 ←────┘
```

**网表**:
```spice
I1 0 1 3.0
r1 1 0 10
D1 0 1 1e-9 27
```

## 解析

与 IRD 相同的拓扑，但二极管反向偏置（阳极接地，阴极接节点 1）。

- 反向偏置时二极管几乎不导通，相当于开路
- 全部 3A 电流流经电阻
- 预期节点 1 电压 = 3A × 10Ω = 30V

这与纯 IR 测试一致，验证了二极管在反偏时的正确行为。

## 验证

```rust
assert_nearly(v[1], 30.0);
```
