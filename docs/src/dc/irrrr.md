# IRRRR — 电流源 + 四电阻梯形网络

**测试文件**: `tests/test_irrrr.rs`
**测试函数**: `test_irrrr`

## 电路

参考 Imperial College MNA 示例。

```
    I1 (3A)
    ┌──→──┐
    │     │
    0     1
    │     │
    │    R1(5Ω)
    │     │
    │     2 ─── R3(10Ω) ─── 0
    │     │
    │    R2(5Ω)
    │     │
    │     3 ─── R4(10Ω) ─── 0
    │     │
    └─────┘
```

**网表**:
```spice
I1 0 1 3.0
r1 1 2 5
r2 2 3 5
r3 2 0 10
r4 3 0 10
```

## 解析

- 节点数：4
- 电压源数：0
- MNA 矩阵维度：4 × 4

纯电阻梯形网络，验证多节点 MNA 求解的准确性。

## 验证

```rust
assert_nearly(v[1], 42.0);  // 节点 1
assert_nearly(v[2], 27.0);  // 节点 2
assert_nearly(v[3], 14.0);  // 节点 3
```
