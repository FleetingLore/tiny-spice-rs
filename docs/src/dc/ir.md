# IR — 电流源 + 电阻

**测试文件**: `tests/test_ir.rs`
**测试函数**: `test_ir`

## 电路

```
    I1 (3A)
    ┌──→──┐
    │     │
    0     │
    │     │
    └──┤R├─┘
       100 (10Ω)
       │
       0
```

**网表**:
```spice
I1 0 1 3.0
R100 1 0 10
```

## 解析

这是最简单的 MNA 测试——一个 3A 电流源驱动 10Ω 电阻。

- 节点数：2（节点 0=地，节点 1）
- 电压源数：0
- MNA 矩阵维度：2 × 2

根据欧姆定律：
$$V_1 = I \\times R = 3 \\times 10 = 30\\text{V}$$

## 验证

```rust
assert_nearly(v, 30.0);
```
