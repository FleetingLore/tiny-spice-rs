# IRDVV — 零伏源电流测量

**测试文件**: `tests/test_irdvv.rs`
**测试函数**: `test_ird_vv`

## 电路

```
    I1 (3A)
    ┌──→──┐
    │     │
    0     1
    │     ├── Vmeas_R(0V) ── R1(10Ω) ── 0
    │     │
    │     └── Vmeas_D(0V) ── D1(→) ── 0
    │
    └──────────────────────┘
```

**网表**:
```spice
I1 0 1 3.0
Vmeas_R 1 2 0
R1 2 0 10
Vmeas_D 1 3 0
D1 3 0 1e-9 27
```

## 解析

这是 SPICE 中常见的**电流测量技巧**：插入一个 0V 电压源来测量支路电流。

- `Vmeas_R`：测量流经电阻支路的电流
- `Vmeas_D`：测量流经二极管支路的电流
- 两个 0V 源不改变电路行为，但在 MNA 中增加额外的未知量（支路电流）

3A 电流源被两条并联支路分流。二极管正向导通，与 10Ω 电阻竞争分流。

assert_nearly(v[1], 0.73217);  // 二极管节点电压
